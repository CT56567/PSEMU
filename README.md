# Goals
- [X] Emulate R3000A PS1 CPU
- [ ] Convert PS1 GPU instructions into code so it can show in a window in opengl
- [ ] Add PS1 BIOS Support
- [ ] Add support for PS1 Custom Sound Chip
- [ ] Add Linux Support
- [ ] Add Wifi Support
